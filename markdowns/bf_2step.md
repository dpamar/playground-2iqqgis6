memory area common

memory area part 1
>                                  array separator
memory area part 2

>[>]>[.>]                          print third memarea

Only the last few chars change... all mem areas can now be generated using the same memory generator tool implemented earlier

* 1st specific part : second skeleton, last line
* 2nd specific part : first skeleton, last line
